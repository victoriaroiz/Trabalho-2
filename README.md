-- Ferramentas de Desenvolvimento e Compilação
Todos os programas dos Problemas 1,2,3  foram desenvolvidos em linguagem C, compilados utilizando MinGW (Compilador GCC) e testados usando plataforma Windows 8.1 via IDE Codeblocks 13.12

-- Compilação
Para compilação do código via Codeblocks, basta utilizar o compilador interno clicando em Build.
O código foi testado em Linux e pode ser compilado da seguinte maneira:

gcc problemaX.c -o problemaX

Substituindo X pelo numero do problema selecionado.

OBS: PARA UTILIZAR ENTRADAS COM ACENTO DEVE SER USADO LINUX

-- Execução
Problema 1:

Ao rodar o programa o seguinte menu é oferecido para o usuário que pode operá-lo indefinidamente até sair:
	(1) Inserir - Usuário insere um numero de elementos e em seguida insere os valores para serem adicionados na arvore
	(2) Remover - Usuário digita uma chave e caso exista ela é removida da arvore
	(3) Impressao Pre-Ordem
	(4) Impressao Pos-Ordem
	(5) Impressao Em Ordem
	(6) Impressao em labelled bracketing
	(7) Busca - Usuário digita uma chave e é informado se ela existe
	(8) Sair - sai do programa
